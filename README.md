Данный репозиторий содержит отчетную документацию, конфигурационные файлы и скрипты, созданные в ходе производственной практики. Основная цель работы — развертывание, настройка и тестирование комплекса программного обеспечения для корпоративного использования.

## 🛠 Используемые технологии

В ходе практики были задействованы следующие инструменты и программное обеспечение:

*   **Операционная система:** Windows Server 2022, Windows 11.
*   **Виртуализация:** Virtual Box.
*   **Служба каталогов:** Active Directory Domain Services (AD DS).
*   **Веб-среда:** XAMPP (Apache, MariaDB, PHP, Perl).
*   **Системы управления контентом (CMS):** WordPress.
*   **Система дистанционного обучения (LMS):** Moodle.
*   **Платформа для разработки:** 1С:Предприятие 8.3 (учебная версия).
*   **Контроль версий:** Git, GitLab.
*   **Средства мониторинга и защиты:** LibreHardwareMonitor, Антивирус Касперского (KES 12).

## ✨ Основные результаты работы

За время практики были выполнены следующие задачи:

1.  **Установка и настройка ПО:** Развернуты и сконфигурированы Windows Server 2022, Active Directory, веб-среда XAMPP, CMS WordPress и LMS Moodle.
2.  **Обеспечение доступа:** Созданы локальные пользователи с разграничением прав доступа на основе принципа минимальных привилегий.
3.  **Контроль совместимости:** Обеспечена корректная работа всех компонентов в единой инфраструктуре без конфликтов.
4.  **Мониторинг и тестирование:** Проведена проверка работоспособности всех систем, проанализированы условия эксплуатации на виртуальной машине.
5.  **Реализация защиты:** Настроены политики безопасности, включая NTFS, антивирусную защиту и брандмауэр.
6.  **Автоматизация:** Предложены сценарии автоматизации резервного копирования с использованием PowerShell.
